# Solar Energy Smart Dashboard 🌞

## Features
- Predict energy consumption
- Compare solar vs usage
- Alerts when solar is insufficient
- Visualization with charts

## Tech Stack
- Backend: Flask (Python)
- Frontend: React.js
- ML Model: Linear Regression

## Run Backend
cd backend
pip install -r requirements.txt
python app.py

## Run Frontend
cd frontend
npm install
npm start
